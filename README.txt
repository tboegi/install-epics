This shell scripts installa EPICS on e.g. a raspberry PI.

The original instructions can be found here:
http://prjemian.github.io/epicspi/


The EPICS home page is here:
EPICS: http://www.aps.anl.gov/epics


The SW image on the raspberry is RASPBIAN, it could be found here:
http://downloads.raspberrypi.org/raspbian_latest

install-epics.sh should even work on a PC running Linux
e.g. Debian, Ubuntu, Scientific Linux.

install-epics.sh should even work under Mac OS X.
